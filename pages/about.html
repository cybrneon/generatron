WHAT IS A GEOMETRY?

In our world, most objects are three-dimensional. A point, a line, a plane, a sphere, and so on. But what about higher-dimensional objects? In our world, there are three dimensions of length, width, and height.
What happens if we add a fourth dimension? Well, there’s a dimension of time. The one thing that we don’t have is a dimension of color.
How do we know that there are only three dimensions of length, width, and height? Well, if we imagine a cube, that’s a perfect three-dimensional object.

GENERATRON

CYBRNEON proudly presents GENERATRON, an open-source and free audiogenic visual generator. The GENERATRON software allows the user to generate visuals using pre-recorded sound tracks. This allows the user to create a new visuals using 3D shapes for each genre that can later be shared with friends and family or even sell as NFTs. The GENERATRON software will soon allow the user to create their own sound tracks with the user’s choice of audio and visual samples.
